Cómo ejecutar el proyecto del sistema de gestión de salones de belleza (BSMS)

1. Descarga el archivo zip

2. Extraiga el archivo y copie la carpeta bsms

3. Pegue dentro del directorio raíz (para xampp xampp/htdocs, para wamp wamp/www, para lamp var/www/html)

4. Abra PHPMyAdmin (http://localhost/phpmyadmin)

5. Crea una base de datos con el nombre belleza

6. Importe el archivo beauty.sql (dentro del paquete zip en la carpeta de archivos SQL)

7.Ejecute el script http://localhost/bsms (frontend)

How to run the Beauty Salon Management System (BSMS) Project

1. Download the zip file

2. Extract the file and copy bsms folder

3.Paste inside root directory(for xampp xampp/htdocs, for wamp wamp/www, for lamp var/www/html)

4. Open PHPMyAdmin (http://localhost/phpmyadmin)

5. Create a database with name beauty

6. Import beauty.sql file(given inside the zip package in SQL file folder)

7.Run the script http://localhost/bsms (frontend)
